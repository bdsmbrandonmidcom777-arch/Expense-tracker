## Описание программы
Expense Tracker — это графическое приложение на Python (Tkinter) для отслеживания личных расходов.  
Позволяет добавлять записи с суммой, категорией и датой, сохранять их в JSON-файл, просматривать историю, фильтровать по категории и периоду, а также подсчитывать общую сумму расходов за выбранный период.

## Требования для запуска
- Python 3.8 или выше  
- Все используемые библиотеки (`tkinter`, `json`, `os`, `datetime`) входят в стандартную поставку Python, дополнительная установка не требуется.
