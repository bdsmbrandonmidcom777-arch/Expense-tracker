# Expense-tracker
Express Tracker — GUI-приложение на Python (Tkinter) для учёта личных расходов. Добавление записей (сумма, категория, дата) с валидацией, сохранение в JSON. Фильтрация по категории и периоду, подсчёт суммы. Проект содержит тесты (позитивные, негативные, граничные), README и .gitignore.
